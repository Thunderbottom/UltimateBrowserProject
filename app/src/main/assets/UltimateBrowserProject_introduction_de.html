UltimateBrowserProject ist ein Browser, der dafür ausgerichtet ist, die Arbeit, anders, als alle anderen Browser zu erledigen. Zurzeit werden folgende Funktionen unterstützt.

Benutzerdefinierte Startseite:

  • Das erste mal, wenn Sie den Browser starten, wird die Webseite about:blank angezeigt.

  • Benutzen Sie das Menü Zur Startseite hinzufügen, um Webseiten zu merken.

  • Benutzen Sie das Menü Anpassen, um Ihre Startseite zu individualisieren.

  • Halten Sie eine Karte auf der Startseite gedrückt, um sie zu bearbeiten.

Neu!

Benutzerdefinierter User-Agent:

Nun können Sie Ihren eigenen User-Agent einstellen. Für eine Liste von User-Agents, besuchen Sie die Webseite User Agent List und setzen Sie es unter Einstellungen -> User-Agent -> Benutzerdefiniert ein.

Zwischen Tabs wechseln:

  • Sie können die Tabswitcher-Position auf oben oder unten festlegen, indem Sie die Einstellung unter Einstellungen -> Browser -> Position des Tabswitchers ändern.

  • Press the address bar and drag it down or up, then the tab switcher will be displayed.

  • Swipe up/down to dismiss a page.

Denken Sie daran, dass wenn die Tastatur offen ist, dass der Tabswitcher nicht angezeigt wird. Dies ist kein Fehler, sondern so vorgesehen.

Im Hintergrund laden:

  1. UltimateBrowserProject als Standardbrowser festlegen.

  2. Eine einzigste Berührung öffnet Links im Hintergrund, und zeigt eine Benachrichtigung in der Benachrichtigungsleiste.

  3. Den Link doppelt tippen öffnet ein Dialog, um den Link im Vordergrund zu öffnen.

Lautstärketasten:

Unter Einstellungen -> Browser -> Lautstärketasten können Sie das Verhalten festlegen auf:

  • Zwischen Tabs wechseln.

  • Durch die Webseite scrollen.

  • Systemstandard.

Neu!

Wischen, um zwischen Tabs zu wechseln:

Nun können Sie die Omnibox (Adressleiste) verwenden, um zwischen Tabs zu wechseln!

AdBlock (Werbeblocker) Whitelist:

Da AdBlock fehlerhafte Webseitendarstellung hervorrufen kann, können diese Seiten zur Whitelist unter Einstellungen -> Werbeblocker -> Whitelist eingestellt werden.

Readability (Lesbarkeitsmodus):

UltimateBrowserProject unterstützt den Lesbarkeitsmodus (Readability), aber benötigt einen Readability API Token.

Um den Token zu erhalten, muss über die Webseite Readability Developer APIs ein Token angefordert werden und unter Einstellungen -> Lesbarkeitsmodus -> Token eingestellt werden.

Screenshot:

UltimateBrowserProject kann ganze Webseiten aufnehmen!.

Leider kann damit nicht die ganze Seite aufgenommen werden, da sonst nicht genügend Arbeitsspeicher vorhanden ist (OOM).

Protokollierung:

UltimateBrowserProject kann automatisch Protokolle senden, wenn die App abstürzt.

Sobald die App abstürzt, erscheint ein Dialog zum Absenden des Berichts. Sie können auch einen Kommentar hinzufügen (OPTIONAL), und dann OK drücken.

Alles andere wird in der Anwendung selber geregelt!
